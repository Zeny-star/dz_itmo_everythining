
\section{Моделирование}

Для моделирования использовался язык программирования python, с использованием пакетов numpy, matplotlib, scipy. Для решения дифференциальных уравнений использовался метот DOP853, {https://en.wikipedia.org/wiki/Runge%E2%80%93Kutta_methods}, основная аппроксимация имеет порядок 8, этого достаточно, так как мы смотрим на смещения порядка $10^-3$ м. Также метод включает механизм оценки ошибок, что дает нам малую ошибку, порядка $O(h^8)$. При помощи метода Рунге-Кутты, на каждом шаге решается дифференциальное уравнение на основе текущего значения $y(t)$, временного шага $h$, и функции $f(t,y)$, описывающей уравнение, значение $y(t+h)$ вычисляется с использованием порядка 8, оценивается ошибка с использованием формул порядка 5 и 3, чтобы адаптировать шаг$h$, если ошибка выше заданного порога, шаг уменьшается, если ошибка мала, шаг увеличивается для ускорения вычислений.
