1) Для решения дифф-х уравнений в данной работе используется метод Рунге-Кутта 8-го порядка.
2) На первом слайде изображены графики с начальными условиями, угол phi=45, и без начальных скоростей. На первом графике изображена зависимость конечного отклонения по оси x от начальной высоты. На втором графике изображены зависимости конечного отклонения по оси y в зависимости от начальной высоты, зеленый график - численное решение, красное - аналитическое. На третьем графике показана разность между аналитическим решением и численным по оси y, на 4-м графике по оси x.
3) На втором слайде изображены графики с начальными условиями, угол phi=45, начальная скорость по оси x=0.1м/с, начальная скорость по оси y = 0. Видно, что график меняет выпуклость, понятно что это происходит из-за того, что теперь основной вклад в отклонение вносит скорость по соответствующеий оси. Остальные графики почти не изменились, кроме изменения разности между приближением и \delta x, понятно что она возрастет тк точность решения дифф-х уравнений упадет.
4) На третьем стайде изображены графики с начальными условиями, улол phi=90, начальная скорость по оси x=0.1, начальная скорость по оси y=0. Очевидно, что на полюсе нету смещения по оси y и x, даже из простых соображений симметрии. Остальные графики без изменений.
